Working with file name

As Wordpress is mainly used to move files, having some functions to work with file names can be a very good idea!

val into : Yocaml.Aliases.filepath -> Yocaml.Aliases.filepath -> Yocaml.Aliases.filepath

filepath |> into dir describes a filepath into a dir.

val with_extension : string -> Yocaml.Aliases.filepath -> bool

with_extension ext path returns true if path ends with ext false otherwise, ie: with_extension "html" "index.html" returns true but with_extenstion "html" "foohtml" returns false.

val basename : Yocaml.Aliases.filepath -> Yocaml.Aliases.filepath

Keep the filename and remove the path.

val add_extension : Yocaml.Aliases.filepath -> string -> Yocaml.Aliases.filepath

Add an extension to a filepath. For example: add_extension "index.txt" "html" will produce "index.txt.html".

val remove_extension : Yocaml.Aliases.filepath -> Yocaml.Aliases.filepath

Remove the extension of a filepath.

val replace_extension : Yocaml.Aliases.filepath -> string -> Yocaml.Aliases.filepath

Replace the extension of a filepath.
